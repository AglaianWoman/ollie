Ollie

Open Information Extraction Software

About

Ollie is software that automatically identifies and extracts binary relationships from English sentences. Ollie is designed for information extraction, where the target relations cannot be specified in advance and speed is important.

To get a better idea of what Ollie does:

  • Download the latest binaries
  • Read Open Language Learning for Information Extraction

Code

Ollie is written in the Scala programming language and is released under the full GPL3 because it depends on Stanford CoreNLP. For instructions on how to run Ollie or use it in your own code, please see the README in the source distribution.
